# GRE 3000 词离线版

一款按照原书词序学习 GRE 词汇的桌面应用。程序以 List 为学习单位，提供阅读、简义、回忆和四选一四种模式，并支持 0～3 星标注、等价词、同词根词、近形异义词和机经 7.0 重点标记。

> 当前稳定版支持 Windows 10/11。iPhone 不能运行 `.exe`；适配 iPhone 的 PWA 网页版与跨设备进度同步方案见下文“在 iPhone 上使用”。

## Windows 快速开始

### 1. 下载

从本项目的 [Releases](https://github.com/KumamuKuma/gre-3000-offline/releases) 页面下载：

```text
GRE 3000 词离线版.exe
```

程序是单文件版本，无需安装，也不需要一直联网。当前构建的 SHA-256 为：

```text
9a0317a4e1541b70661b946c7c468d763ab5e711f5924f3534efa3cb0d799b0a
```

在 PowerShell 中可以这样核对下载文件：

```powershell
Get-FileHash -Algorithm SHA256 ".\GRE 3000 词离线版.exe"
```

### 2. 启动

双击 EXE 即可启动。因为这是未购买代码签名证书的个人构建，Windows SmartScreen 首次运行时可能提示“Windows 已保护你的电脑”。请先确认文件来自本仓库并核对 SHA-256，然后选择“更多信息”→“仍要运行”。

### 3. 开始学习

1. 在首页选择要学习的 List。
2. 选择“全部单词”或指定星级。
3. 选择学习模式并点击开始。
4. 使用“上一词/下一词”按原书顺序学习。
5. 在学习页右上角点击星星，为当前单词标注 0～3 星。
6. 完整走到 List 最后一个词后，点击“完成本轮”，该 List 的已背次数会增加一次。

## 学习模式

| 模式 | 显示方式 | 适合场景 |
| --- | --- | --- |
| 阅读模式 | 直接显示完整释义、近义词和例句 | 初次学习 |
| 简义模式 | 直接显示精简的英中释义 | 快速过词 |
| 回忆模式 | 先隐藏释义，点击或按空格后揭晓 | 主动回忆 |
| 四选一模式 | 从四个词义中选择正确答案 | 自测巩固 |

切换模式不会自动跳到另一个单词。回忆模式揭晓前、四选一作答前会隐藏容易泄题的词根、等价词和近形词提示。

## 主要功能

- **按 List 学习**：可主动选择 List 1～30 或补充 List，每个 List 分别记住当前位置。
- **原书词序**：学习和完整词表均保持原书顺序。
- **星级标注**：每个单词可标为 0、1、2 或 3 星；3 星后再点一次回到 0 星。
- **按星级学习**：可在某个 List 内只背指定星级的单词。
- **List 已背次数**：完成整轮后自动增加，也可以在首页用 `− / +` 手动修正。
- **词汇关系**：在词条下方显示经过审核的《真经 GRE 等价词》关系、同词根/同族词和拼写相近但词义不同的词。
- **机经重点**：同时出现于《GRE 镇考机经词 7.0》的词会显示橙色重点标记。
- **完整词表与搜索**：按词头搜索、查看所属 List、检查重点标记并直接修改星级。
- **单词朗读**：点击“朗读”或按 `P`；设置中可以开启“切换到下一词时自动朗读一次”。

### 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl+F` | 聚焦首页搜索框 |
| `←` / `→` | 上一词 / 下一词 |
| `Space` | 在回忆模式中显示或隐藏答案 |
| `P` | 朗读当前单词 |

在搜索框等可编辑控件中输入时，学习快捷键不会抢占输入。

## 数据保存与备份

学习进度、List 完成次数、星级和设置保存在：

```text
%APPDATA%\GRE Vocab Offline\GRE 3000 词离线版\user.db
```

运行日志保存在同一目录下的：

```text
logs\app.log
```

更新程序时，关闭旧版本并用新 EXE 替换即可；个人进度不会被覆盖。需要备份时，先完全退出程序，再复制 `user.db`。删除整个数据目录会重置个人数据。

## 在 iPhone 上使用

Windows EXE 无法直接安装到 iPhone。这个项目选择 **PWA（可安装网页应用）** 作为移动端方案，而不是先做微信小程序。

### 为什么选择 PWA

- Safari 打开网址后即可“添加到主屏幕”，使用体验接近普通 App。
- 不需要 Apple 开发者账号，也不需要等待 App Store 或微信审核。
- 可以共用同一套词库和学习逻辑，同时适配 iPhone、iPad、Mac 和其他电脑。
- 支持离线缓存；首次载入后，无网络时仍可继续学习。
- 更适合后续加入 Windows 与 iPhone 的双向进度同步。

### 计划中的同步方式

移动端会使用账号登录，并把下面这些个人数据保存到私有云端：

- 各 List 当前学习位置；
- 各 List 已背次数；
- 每个单词的 0～3 星标注；
- 自动朗读等个人设置。

Windows 版和 PWA 会使用同一账号同步。词库内容保持只读，个人进度不会提交到 GitHub，也不会公开给其他用户。离线时产生的修改会先保存在设备本地，恢复网络后再合并。

### PWA 完成后的安装方法

1. 用 iPhone 的 Safari 打开 PWA 地址。
2. 点击底部“分享”按钮。
3. 选择“添加到主屏幕”。
4. 从主屏幕打开 GRE 3000，登录后即可同步学习进度。

> 当前仓库中的 iPhone PWA 和云同步尚未发布，Releases 中的 EXE 仍然只适用于 Windows。移动端上线后，此处会补充正式网址。

## 开发与验证

桌面端使用 Python 3.12、PySide6 和 SQLite。安装开发依赖后运行测试：

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -e ".[dev]"
$env:QT_QPA_PLATFORM = "offscreen"
.\.venv\Scripts\python.exe -m pytest -v -p no:cacheprovider
```

重新生成词库时，还需要通过环境变量提供三份合法取得的参考 PDF：

```powershell
$env:GRE_SOURCE_PDF = "<张巍 GRE 镇考 3000 词 PDF 路径>"
$env:GRE_EQUIVALENCE_PDF = "<真经 GRE 等价词汇总 PDF 路径>"
$env:GRE_MACHINE7_PDF = "<GRE 镇考机经词 7.0 PDF 路径>"
```

构建 Windows 发布版：

```powershell
powershell -ExecutionPolicy Bypass -File scripts/build_release.ps1
```

发布脚本会执行全量测试、严格词库导入、数据库与审计核对、图标生成、单文件构建和原生窗口启动检查。全部通过后才会更新 `outputs` 中的 EXE、使用说明和词库审计报告。
