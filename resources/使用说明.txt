GRE 3000 词离线版｜使用说明
============================

一、启动
双击“GRE 3000 词离线版.exe”即可启动，无需安装，也不需要联网。程序启动时会临时解包运行组件，正常关闭后自动清理。

这是未签名的个人构建，Windows SmartScreen 可能在首次运行时提示“Windows 已保护你的电脑”。请先确认文件来源和发布者提供的 SHA-256；确认无误后，可点“更多信息”继续运行。

二、本地数据位置
各 List 的学习位置与完成次数、星级和设置保存在：
%APPDATA%\GRE Vocab Offline\GRE 3000 词离线版\user.db

运行日志保存在同目录的 logs\app.log。词库随 EXE 只读发布，不会覆盖 user.db。更新版本时，关闭程序后用新 EXE 替换旧 EXE 即可，原有 List 进度和星级会保留。旧版 4、5 星会自动合并为 3 星；旧收藏和逐词已背次数不再显示。若主动删除上述数据目录，个人数据将被重置。

三、学习方式
1. 在首页主动选择 List 1–30 或两个补充 List；每个 List 独立记住位置，全部按原书词序。
2. 可按星级学习：选择 0、1、2 或 3 星后，可任意勾选多个 List，也可以选择全部 List。符合条件的单词会合并后严格按原书顺序呈现；所选范围及当前位置会随进度同步。选择“全部星级”时仍按上方单个 List 学习。没有匹配单词时开始按钮会禁用。
3. “阅读模式”直接显示完整释义、近义词和例句。
4. “简义模式”直接显示精简英中释义，不需要点击。
5. “回忆模式”先隐藏答案，按空格显示或隐藏精简英中释义。
6. “四选一模式”提供四个词义；选择后会标出正确答案，但不会自动跳词。学习页可随时独立开关“答错 +1 星”和“答对 −1 星”，只在每个单词首次作答时自动调整，星级始终限制在 0 至 3 星。
7. 各学习和单词详情界面右上角均可标注 0 至 3 星。每点一次增加一星，3 星后再次点击回到 0 星。
8. 完整学习到 List 最后一个词后，点击“完成本轮”会给该 List 增加一次已背记录；星级筛选学习不计为完整一轮。首页所选 List 旁的“− / +”可手动修正次数。
9. “完整词表”按原书顺序显示全部单词和所属 List，可筛选、查看机经 7.0 重点标记并修改星级。
10. 可可靠识别的同词根/同族词和拼写相近但词义不同的词会显示在单词下方，并可点击打开。回忆揭晓前和四选一作答前会隐藏这些提示。
11. 《真经 GRE 等价词汇总》中直接列明且能与主词库精确匹配的等价词会显示在词条下方；关系双向可见，但不会把没有直接来源依据的词自动串成一组。
12. 同时收录于《GRE 镇考机经词 7.0》的单词会显示橙色“机经 7.0 重点词”标记，完整词表中也会显示“重点”。

四、快捷键
Ctrl+F       聚焦首页搜索框
← / →        上一词 / 下一词
Space        回忆模式中显示或隐藏答案
P            朗读当前单词

在搜索框等可编辑控件中输入时，学习页快捷键不会抢占输入。

五、英文语音
朗读功能有三种状态：
1. 已检测到英文语音：可在“设置”中选择英文语音并调整速度。
2. 未检测到英文语音，但系统默认语音可用：程序会使用系统默认语音，并提示建议安装英文语音包；朗读效果和发音语言取决于该系统语音。
3. 未检测到可用语音引擎：此时仅禁用朗读；搜索、学习、星级和进度保存仍可正常使用。

可在“设置”中打开“切换到下一词时自动朗读一次”。该开关会在进入下一个词或使用 List 开头/结尾跳转后朗读当前词；首次打开 List 不会自动播放。

学习页底部提供“到开头”和“到结尾”，可一键跳到当前 List（或当前星级筛选结果）的第一个或最后一个单词。

可在 Windows 的语言/语音设置中另行安装英文语音包。

六、iPhone 网页版与进度同步
网页版地址：https://gre-3000-offline.cbg206.chatgpt.site

在 iPhone Safari 中打开后，点“分享”→“添加到主屏幕”即可像 App 一样启动。首次在线载入后支持离线学习。

手动同步：在 Windows 或网页版的“设置”中导出 JSON 进度文件，再到另一端导入。

免账号云同步：在网页版“设置”中点击“创建新同步码”，复制以 GRE1- 开头的同步码；然后在 Windows“设置”→“免账号同步码”中粘贴同一组码。可点击“上传本机进度”或“从云端恢复”。进度会先在设备端使用 AES-256-GCM 加密，服务器仅保存密文。同步码就是身份凭证和解密钥匙，请勿公开并妥善备份；遗失后无法找回。
